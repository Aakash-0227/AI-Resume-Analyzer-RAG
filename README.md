# AI Resume Analyzer with RAG

An AI-powered Resume Analyzer that uses Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs) to analyze resumes against job descriptions.

## Features

- ATS Score Analysis
- Resume Summarization
- Skill Gap Analysis
- Job Match Percentage
- Missing Keyword Detection
- Interview Question Generation
- Resume Q&A using RAG

## Tech Stack

- Python
- LangChain
- FAISS
- Hugging Face Embeddings
- Gemini API
- Streamlit
- PyPDF2
